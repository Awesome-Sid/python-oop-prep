# 10 — Advanced Python OOP

Topics senior screens often touch:

- **`@dataclass`** — boilerplate `__init__`, `__repr__`, optional ordering
- **`__slots__`** — fixed attributes, less memory, no `__dict__` unless added
- **`Enum`** — named constants (browser types, env)
- **Descriptors** — `@property` is a descriptor; advanced frameworks use custom ones
